* LoRA: Low-Ran Adaptation of LLMs
  * Reduced the number of params to train by 10000x less, 3x less GPU memory
    with only reduced accuracy to finetuning.
  * LoRA basically freeze "main" weights while training new ones.
  * New weights: Rank decomposition matrices of original weight's change.
  * At inference time, trained weights are merged with main weights.

## Hierarchical Navigational Small World Search (Index) method: 

* Nearest Neighbor Vector is a problem with many (thousands) of vectors. HNSW to the rescue:
* HNSW is based on: Hierarchical Navigational Small World:<br/>
 Observed phenomenon in human social networks where everyone is closely connected ("small world").
  * Six Degrees of Separations: Popularized concept that any two individuals are on average separated by six acquaintance links.
  * High Clustering: Nodes from tightly-knit groups, with members of groups connecting to one another.
  * Short Path Lengths: Despite hight clustering, only a few steps a re needed to connect any two nodes.

   Ente.Data Store -> LLM : Results

   App -> LLM : Q
   LLM -> External Processing : Command
   External Processing -> LLM :  structured Data
   LLM -> App : Answer

   App -> LLM : Q
   LLM -> External Data Store: Search 
   External Data Store -> External Processing: ..
   External Processing -> LLM: Structure data
