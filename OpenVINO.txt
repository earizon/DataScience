# Intel OpenVINO

* OpenVINO toolkit stands for Open Visual Inference and Neural Network Optimization.

* It supports GenAI and conventional AI models.
* For GenAI it provides LLM-specific interfaces for easy integration of 
  language models, handling tokenization and text generation.

* "Lower layer" supporting PyTorch, TensorFlow, TensorFlowLite, PaddlePaddle,  
  ONNX, Keras and  JAX (As of 2026). <br/>
  
* Supported Intel CPUs: Atom/Core/Core Ultra/Xeon/ARM 
* Supported Intel GPUs: IRIXMax,  ARC, DataCenter GPU
* Supported Intel NPUs: Core Ultra.
* Supported Intel FPGA: "AI Suite".


* designed for Intel's new generation of "Meteor Lake" ("Intel Core Ultra") 
  mobile CPUs herald a new age of "AI PCs," (2024) featuring an NPU (neural 
  processing unit) that's purpose-built to handle AI tasks.  In support 

* Can be installed through PIP virt.environemts, tarballs, APT/YUM, Docker, Conda,
  Zypper (C++ API only), ...

OpenVINO plugins exists also for different apps like:

* (OpenVINO Stable Diffusion) Generate images in GIMP.
* (OpenVINO Music Generation): text (prompt)-to-(simple)music in Audacity.
* (OpenVINO Whisper Transcription): Transcribes your audio to text in Audacity.
* (OpenVINO Noise Reduction): Removes background noise in Audacity.
REF: <https://www.tomshardware.com/laptops/i-tested-intels-meteor-lake-cpus-on-ai-workloads-and-amds-chips-sometimes-beat-them>

## OpenVINO GenAI vs PyTorch, TensorFlow, JAX, ...

* PyTorch, TensorFlow, JAX, ... offer general programming libraries for model training, matrix
  optimizations in GPUs, general APIs "a la NumPy", ...
* OpenVINO GenAI focuses in executing the inference phase of Deep-Neural Networks previously
  architected and trained with PyTorch, TensorFlow, JAX...

* PyTorch, TensorFlow, JAX, ... have different "low level" layers to interact with the hardware
  like NVidia CUDA, Intel OpenVINO, ...
* OpenVINO GenAI is a library of the most popular Generative AI model pipelines, optimized
  execution methods, and samples that run on top of highly performant OpenVINO Runtime.
